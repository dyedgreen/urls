Urls Bot

Website metadata crawler

This website will crawl urls submitted by it's users for meta information such as title, or description. This is accomplished by making an automated HTTP request, and then parsing the returned HTML document.

The bot supports the standard HTML <title>title</title> tag, as well as Twitter cards and Facebook Open Graph.

The code for the parser used can be found on GitHub. It's user-agent is:

Mozilla/5.0 (compatible; Urlsbot/0.1.0; +https://urls.fyi/bot.html)
